B핵심 아이디어: 회전 불변량

"손가락이 펴졌다"를 방향 없이 정의하려면? 손끝이 손목에서 얼마나 멀어졌는지를 보면 됩니다:

  펴진 손가락              굽힌 손가락
      ● tip                   ● pip
      │                      ╱ ● tip  ← 접혀서 손목 쪽으로
      ● pip                 │
      │                     │
  ────●──── wrist       ────●──── wrist

  ratio = dist(tip, wrist) / dist(pip, wrist)
  펴짐: ratio ≈ 1.25+     굽힘: ratio ≈ 0.95-
  ★ 손을 어떻게 돌려도 두 거리는 같이 돌아간다 → 비율은 불변
왜 절대거리가 아니라 비율? 절대거리는 손 크기·카메라와의 거리에 오염됩니다. 비율은 그 오염이 분모·분자에서 상쇄 — 정규화의 기하학 버전. 그리고 z까지 3D로 계산해서 손가락이 카메라 쪽으로 접혀도 잡아냅니다.

판정 규칙은 펴진 손가락 개수(엄지 제외 4개 중):

펴진 손가락판정
0개✊ 바위
딱 2개 = 검지+중지✌ 가위 (중지+약지 2개는 가위 아님)
4개✋ 보
그 외 (1·3개, 애매). . . 판정 보류

Script CHOPscript_fingers — ★새 개념: 파이썬으로 만드는 CHOP

Script CHOP은 매 프레임 파이썬 함수(onCook)를 실행해 채널을 직접 만드는 오퍼레이터입니다. 1번 입력에 select_h1, 2번 입력에 select_active를 와이어. 코드 전문:

import math

TIP = {'index': 'index_finger_tip', 'middle': 'middle_finger_tip',
       'ring': 'ring_finger_tip',   'pinky': 'pinky_tip'}
PIP = {'index': 'index_finger_pip', 'middle': 'middle_finger_pip',
       'ring': 'ring_finger_pip',   'pinky': 'pinky_pip'}

R_CURL, R_OPEN = 0.95, 1.25   # 굽힘/펴짐 경계 — 캘리브레이션 노브

def d3(src, a, b):
    # 두 랜드마크 사이 3D 거리 (x·y·z 전부)
    return math.sqrt(sum(
        (src['h1:%s:%s' % (a, ax)][0] - src['h1:%s:%s' % (b, ax)][0]) ** 2
        for ax in 'xyz'))

def onCook(scriptOp):
    scriptOp.clear()
    src = scriptOp.inputs[0]
    active = scriptOp.inputs[1]['h1:hand_active'][0]

    ext = {}
    for f in TIP:                                  # 손가락 4개 각각
        d_tip = d3(src, TIP[f], 'wrist')
        d_pip = d3(src, PIP[f], 'wrist')
        ratio = d_tip / d_pip if d_pip > 1e-6 else 0.0
        # 비율 → 0~1 "펴짐 정도" (같은 척추: 정규화!)
        ext[f] = max(0.0, min(1.0, (ratio - R_CURL) / (R_OPEN - R_CURL))) * active

    up = [f for f in ext if ext[f] > 0.5]          # 펴진 손가락 목록
    n = len(up)
    rock     = active if n == 0 else 0.0
    paper    = active if n == 4 else 0.0
    scissors = active if (n == 2 and 'index' in up and 'middle' in up) else 0.0

    for name, val in list(ext.items()) + [('count', float(n)),
            ('rock', rock), ('paper', paper), ('scissors', scissors)]:
        c = scriptOp.appendChan(name)              # 채널을 직접 만든다
        c.vals = [val]
읽는 순서. ① d3 = 피타고라스 3D 거리 ② 손가락마다 tip/pip 비율 ③ 비율을 0~1로 정규화 (Math CHOP이 하던 일을 코드로) ④ 0.5 넘는 손가락 세기 ⑤ 개수로 판정 ⑥ appendChan으로 채널 출력. hand_active를 곱해 손이 없으면 전부 0 — 06에서 배운 게이트 패턴.
