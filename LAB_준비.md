### 설계 원칙 두 개

```
① 공통 척추 = 입력 → 정규화 → 매핑
   미디어파이프든 아두이노든 똑같은 사고방식. 소스만 (웹캠 ↔ 센서) 바뀜.
   → 두 번째 센서 날이 "새 개념"이 아니라 "복습 + 확장"이 됨

② 실력 섞임 대응 = 가이드 빌드 → 자유 확장
   14:30 가이드 빌드(다같이 완주)  → 초보 = 완주 목표
   16:15 자유 확장(고수는 더 깊게) → 고수↔초보 버디 페어
```

## 회차별 상세

### 4회차 (7/9) — TD 기초 + 첫 인터랙션

**목표:** "내 입력이 화면을 흔든다"를 첫날 모두가 체험. 추가 장비 0.

**가이드 빌드 (다같이):**
1. TD 실행 → 오퍼레이터 패밀리 한 장으로 설명
   ```
   TOP  이미지/영상      CHOP  신호·숫자(시간축)
   SOP  3D 점/지오메트리  DAT   텍스트·표
   COMP 컨테이너/UI       MAT   재질
   ```
2. `Video Device In TOP` → 웹캠 화면 띄우기 (첫 성공감)
3. `Edge TOP` → 윤곽선, `Level TOP` → 밝기/대비 장난
4. `Feedback TOP` + `Composite/Over` → 잔상 트레일
5. `Mouse In CHOP` → Transform/Displace 파라미터에 드래그 = 마우스로 화면 흔들기
6. `Audio Device In CHOP` → `Audio Spectrum` → 소리로 파라미터 흔들기
7. ★ 칠판: **와이어 = 데이터 흐름** (모든 인터랙션의 뼈대)

**자유 확장:** GLSL TOP 맛보기 / feedback 네트워크 깊게 / 이펙트 체인 늘리기

**만들 데모:** `04_basics_완성본.toe` + `04_basics_시작본.toe`(빈 캔버스)

**강사 체크:** 웹캠 작동 / TD 설치 사전공지 / 마우스 인 좌표계 설명

**★ 이 날 센서 수요조사지 배포 + 설명**

### 5회차 (7/16) — 미디어파이프 연동 (웹캠 = 몸 컨트롤러)

**목표:** 웹캠으로 손/포즈 트래킹 → "입력→정규화→매핑" 머리 만들기. 장비 0, 집에서 연습 가능.

**가이드 빌드 (다같이):**
1. `mediapipe-touchdesigner` .toe 실행 (설치 불필요, 웹캠 드롭다운에서 선택)
2. Hand 모델 ON → 오버레이로 손 21 랜드마크 확인
3. 출력 CHOP에서 `Select CHOP`으로 검지끝(index tip) x, y 뽑기
4. **정규화:** `Math CHOP`(0~1 범위) → `Lag CHOP`(부드럽게) → `Filter CHOP`
5. **매핑:** 좌표 → `Circle/Particle TOP` 위치 or `Sphere SOP` or 오디오 파라미터
6. 포즈 트래킹으로 전환 → 같은 패턴 그대로 재사용
7. ★ 칠판: **입력 → 정규화 → 매핑** (이 그림이 6회차에도 똑같이 나온다고 예고)

**자유 확장:** 멀티핸드 / 제스처(thumbs up 등) 트리거 / 얼굴 메시 / 배경 제거

**만들 데모:** `05_mediapipe_hand.toe`, `05_mediapipe_pose.toe`

**강사 체크:** 플러그인 Mac/PC 사전 테스트 / 조명 영향 설명 / GPU 부하 주의

### 6회차 (7/23) — 아두이노 시리얼 + 프로젝트 소개  *촬영

**목표:** 5회차와 **똑같은 매핑 패턴**, 소스만 물리세계로. 손맛 + 집에서 계속 놀 수 있음.

**가이드 빌드 (다같이):**
1. Arduino IDE → 보드/포트 선택 → Blink 업로드 (연결 확인)
2. 가변저항 회로 (브레드보드: 양끝 5V·GND, 가운데 → A0)
   ```
   analogRead(A0) → Serial.println(value)   // 0~1023
   ```
3. TD: `Serial DAT` → 포트·보드레이트 설정 → 들어오는 값 확인
4. `DAT to CHOP` → 숫자로 변환 → `Math/Lag CHOP` 정규화
5. **매핑:** 5회차와 동일! 값 → 비주얼/사운드
6. 초음파(HC-SR04)·MPU-6050로 확장 (라이브러리 사용)
7. ★ 칠판: 5회차 그림 다시 → "머리는 똑같지? 소스만 바뀜"

**프로젝트 소개 + 브레인스토밍 + 멘토링 ①** (후반 시간)

**자유 확장:** 센서 2개 동시 / 멀티 아두이노 / 시리얼 여러 값 파싱

**만들 데모:** `06_arduino_pot.toe` + 아두이노 스케치(`pot.ino`, `ultrasonic.ino`, `mpu6050.ino`)

**강사 체크:** 키트 사전 점검 / 드라이버(CH340 등) 설치 안내 / 보드레이트 일치(9600 등)
